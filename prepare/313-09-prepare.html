Code Snippets

Here are a few helpful one liner code snippets that may come in handy:

Getting get/post variables in Servlets:

  • String password = request.getParameter(“password”);

Getting get/post variables in JSPs:

  • This is a URL parameter: ${param.password}

Setting a session variable in a Servlet:

  • request.getSession().setAttribute(“userId”, “test123”);

Getting a session variable in a JSP:

  • This comes from the session: ${sessionScope.userId} or simply: ${userId}

Transferring control from a servlet to a JSP to produce the content for the response:

  • request.getRequestDispatcher(“/myPage.jsp”).forward(request, response);

Getting the current directory of a servlet (on the server), and opening a file for output in that dir:

  • String path = getServletContext().getRealPath("/") + "filename.txt";
  • BufferedWriter out = new BufferedWriter(new FileWriter(path));
