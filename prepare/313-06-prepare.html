Basic query information

Please note the importance of the WHERE clause on the update and delete queries. Without it, you will update or delete every row in the table! It is often helpful to run the exact query you plan to use for a delete as a “SELECT * FROM…” type of query first, to see the rows that will be affected, and then change the “SELECT *” to “DELETE,” etc.

Connecting from PHP

Running these queries from PHP is no different than connecting and running a Select query as we have done previously.

One point that is worth noting: after inserting a row into a table where an auto-generated ID is the primary key, it is often useful to get that new id, for future updates or inserts into other tables. The last inserted id is can be obtained using:

If you are using mysqli:
$newId = $mysqli->insert_id;

If you are using PDO
$newId = $pdo->lastInsertId();
