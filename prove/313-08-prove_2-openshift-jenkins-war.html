Option #3 - Deploying a WAR file using Jenkins.

1. Follow the steps from Option #1 to set up Jenkins.

2. Follow the steps from Option #2 to build a WAR file.

3. Rather than commit the source of your project to GitHub, create a new git repository in an empty directory. Create a webapps subdirectory, and put the .war file in it.

4. Create a .openshift/markers/java7 file as described in Option #1.

5. Using Git, add, commit, and push your files to GitHub.

6. Log into Jenkins and click build now to build your application.
