Submit Your Assignment

After creating a Hello World page and pushing it to both OpenShift and GitHub, submit this assignment by doing the following:

  1. Make sure your latest code is pushed to GitHub.
  2. In I-Learn, navigate to the "Wiki Google Doc." in the "Resources" folder. We will use this Google Doc as a course wiki throughout the semester to post links to our work and any other helpful tips.
  3. Self-alphabetize this list by last name, and include the following:
    • Name
    • Web application URL (The link to your Hello World page at OpenShift)
    • Source control URL (The link to your repository at GitHub)
  4. Download the CS 313 Submission Form text file.
  5. Fill out this form.
    • For the project title, put "Hello World".
    • Make sure the URL for your web application points to your OpenShift server, and shows your Hello World page in a browser.
    • Make sure your source code repository URL points to your GitHub page.
    • Sections like "any special instructions" won't be very important this week, but will be in future submissions.
    • Because this first assignment is pass/fail, for the category that best describes your assignment, select "Meets requirements" and then no justification is needed.
    • Save your changes to the PDF
  6. Upload this PDF to the I-Learn Dropbox for this assignment.
