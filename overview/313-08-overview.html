Java Servlets

Introduction

This week we are changing gears and switching technology from PHP to Java. Specifically we will start off by looking at Java Servlets (which are the foundation of almost all other Java Web frameworks, etc.).

Objectives

The major assignment this week is to configure your environment and infrastructure for future Java Web development work. To make sure everything is in place, we will be doing a simple "Hello World" servlet for the weekly assignment.

Java Projects

For the remainder of the semester we will be working on projects that will guide much of our learning and assignments. This projects will be similar to the PHP projects we did for the first half of the semester, except that they will be require Java and will be more open ended. You will need to have a proposal in place by the end of next week, but by the end of this week you should have a high-level idea of what you want to work on.
