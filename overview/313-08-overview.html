JAVA SERVLETS

INTRODUCTION

This week we are changing gears and switching technology from PHP to Java. Specifically we will start off by looking at Java Servlets (which are the foundation of almost all other Java Web frameworks, etc.).

OBJECTIVES

The major assignment this week is to configure your environment and infrastructure for future Java Web development work. To make sure everything is in place, we will be doing a simple “Hello World” servlet for the weekly assignment.

GROUP PROJECTS

For the remainder of the semester we will be working on group projects that will guide much of our learning and assignments. You will need to have a proposal in place by the end of next week, but by the end of this week you need to have determined your groups and hopefully have a high-level idea of what you want to work on. Please post ideas and interest in different groups to the Course Wiki.
