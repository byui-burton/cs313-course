JAVA DATABASE ACCESS (JDBC)

INTRODUCTION

This week will we be learning about database access in Java. In particular, will we be working with JDBC, which should appear similar in many ways to the database interaction we did in PHP. Next week, we will learn the basics of an Object-Relational Mapping (ORM) approach, which provides a slightly different form of database access by building another layer of abstraction on top of a more fundamental layer like we are learning this week.

OBJECTIVES

As always, there will be preparation material and a team activity to start the week, but then the weekly assignment is simply to complete the next milestone of your group project. Because of the different nature of your projects, this may require some groups to dive deeper into these concepts than others depending on your specific project needs.

Also, even though each group is working on different topics or APIs, please don't forget to contribute to the course wiki regarding something you found.
