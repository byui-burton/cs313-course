PHP DATABASE ACCESS

INTRODUCTION

Last week we discussed retrieving data from a database and displaying it on a PHP page. This week we will build on these ideas by modifying the data.

This week does not present a significant amount of “new material,” but rather is a further application of the principles and topics we have been discussing. The specific topic for the week is modifying data from PHP. In so doing, we need to use insert, update, and delete queries, and then we need to use these queries from PHP and in the context of relations between tables, etc.

OBJECTIVES

In this regard, the weekly assignment is to complete the DB modification component of your web application (inserting, updating, and deleting). This should essentially complete the functionality of your application, but there will be one more week (next week) for you to add any finishing touches and bells and whistles that you would like.
