JAVA SERVER PAGES (JSPs)

INTRODUCTION

Last week, we learned about the basics of writing and deploying a Java web application. This week, we will be covering more detail of how to make a functioning web site using the tools of JSPs (Java server pages) and Servlets. We will see how these tools are similar and how they are different, and discussing best practices of how to combine them to create web applications in the MVC (Model View Controller) design pattern.

OBJECTIVES

For your weekly team activity, you should work with your new project teams, and you will develop a simple application that uses JSPs and servlets.

Then, the weekly assignment for this week is an individual assignment that requires you to use JSPs and servlets to create a simple web forum. Because we have not talked about database interaction from Java yet, we will be storing this data on the file system. This assignment can be challenging, particularly if your Java skills are a little rusty, so please plan accordingly!

Also, this week, you need to submit a proposal for your group project that will drive the assignments for the rest of the semester.
