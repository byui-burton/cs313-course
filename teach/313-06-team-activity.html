TEACH ONE ANOTHER

Last week, we created a database to hold scriptures and a simple website to display them. This week, we will create a database that can hold scriptures and topics, and link scriptures to topics.

INSTRUCTIONS:
  1. Add to your scripture DB from last week. If you no longer have that database, please create a table for scriptures with columns for id, book, chapter, verse, and content.
  2. Create a table called “Topics” with two columns: id and name;
  3. Manually insert into this table the following topics:
    1. Faith
    2. Sacrifice
    3. Charity
  4. There will be a many to many relationship between scriptures and topics, so you need to create another table to link scripture ids to topic ids (and include foreign keys).
  5. Create a PHP page to insert a new scripture along with its topics. (Note that you are inserting a new scripture and adding a link to an existing topic, all in the same POST)
    1. Show a text box for:
      1. Book
      2. Chapter
      3. Verse
      4. Content (The text of the scripture. A text area would probably be the best choice here).
    2. Then have the PHP script generate checkboxes for each topic contained in the DB table.
    3. Have a submit button to insert the scripture into the database, and connect it to the selected topics.
  6. After submitting, you should show a page that lists all the scriptures in the database, each one with it's associated topics.
  7. Using your PHP page, insert the following scriptures into the database along with the applicable topics:
    1. Hebrews 11:4
    2. 1 Corinthians 13:13
    3. Moroni 7:47

PROVE

For this assignment, you will likely produce two to three PHP files. Each member should submit each of these to I-Learn. You need not submit your database.

As a stretch challenge, you may complete the following steps for extra credit.

Stretch Challenge 1: Create another checkbox at the end of the list, with an empty text box next to it. If the user selects this checkbox, add a new topic with the corresponding name to the database in addition to adding the scripture, and the link between them.

Stretch Challenge 2: Have the list of previously entered scriptures along with their topics display on the same page as the “new scripture entry” portion described above. When a new scripture is entered, refresh the page to show the newly entered scripture along with the previous ones.

Stretch Challenge 3: Complete Stretch Challenge 2 without causing a page refresh (i.e., use AJAX to complete the submission and update).

Please indicate in your submission comments if you completed the basic task, and which stretch goals you completed.
