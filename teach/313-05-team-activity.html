Team Readiness Activity 5

Preparation

Before doing this activity, each student should read the weekly preparation material to ensure that the team meeting is as efficient and effective as possible.

Teach One Another

Objective

Your assignment is to create a simple database and query it to display results on a PHP page. For this assignment, you should work on the same database/PHP file, (rather than doing this separately), so you will want to use screen sharing or other collaborative programs to ensure everyone can be involved.

Instructions

Step 1: Create a new database table called "Scriptures" that has the following columns:

  • id
  • book
  • chapter
  • verse
  • content

Step 2: Insert the following scriptures (along with the text of the verse as content) into your database:

  • John 1:5
  • Doctrine and Covenants 88:49
  • Doctrine and Covenants 93:28
  • Mosiah 16:9

Step 3: Create a PHP page to query this database and list the scriptures. Put a heading at the top of the page to display "Scripture Resources". Then for each scripture, display the book chapter and verse in bold, followed by a hyphen and then the content of the scripture in quotes. For example:

Scripture Resources

John 3:16 - "For God so loved the world, that he gave his only begotten Son, that whosoever believeth in him should not perish, but have everlasting life."

Proverbs 3:5 - "Trust in the Lord with all thine heart; and lean not unto thine own understanding."

Prove

Each member of the team should submit a copy of the PHP file to I-Learn. You need not submit your database.

As a stretch challenge, you may complete the following steps for extra credit:

Stretch Challenge 1: Create a PHP form to search for a book and display all the scriptures in your database that match that book. 

Stretch Challenge 2: Push this database and code to one of your servers and get it to work there. 

Please indicate in your submission comments if you completed the basic task and which stretch challenges you completed.
