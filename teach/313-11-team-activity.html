TEACH ONE ANOTHER

For this activity, we will be using JDBC to connect to a MySQL database and display information in a Java web application.

Specifically, consider the case of a list of ancestors in a database. For simplicity, each person should have a first name, last name, and birthday. There should be a relationship between parents and children modeled in the database (many to many). Then on a web page, you should display a list of all the people in the database. Each name should be a link, where if you click on it, a person details page should be display that shows the person, along with their children (if they exist in the database) and their parents (if they exist in the database).

To achieve this goal, start working with the most basic components first, and then add the additional functionality. Consider the following steps:

  1. Ensure that you have MySQL installed on your development machine.
  2. Create a Person table in the database.
  3. Insert some test people.
  4. Create a Java web application with a page to display each person in the database.
    1. Don't forget to include the JAR's for the JDBC connector, as well as the JSTL.
    2. Write a simple query to select all the people from the database and return them.
  5. Make each name on this page a link to a “Person Details” page. At this point, have the Person Details page simply list the name and birthday of the person.

PROVE
