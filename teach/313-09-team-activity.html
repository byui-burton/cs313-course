TEACH ONE ANOTHER

OBJECTIVE: Be able to use servlets and JSPs together to accomplish a simple task.

OVERVIEW: Create a login page (JSP) that accepts a username and password. If a correct username and password is entered, it should direct the user to a “welcome page” showing their username. If an incorrect username and password are entered, it should direct them to an “incorrect password” page.

INSTRUCTIONS:

Create a JSP login page with a username and password box. Submission of the form should post to a login servlet.

  1. Create a login servlet to check for a valid username and password (for the first pass a correct username and password can be hard coded into the servlet's code). If the username and password match, put the username on the session and send the user to a welcome page, if not send them to an incorrect login page.
  2. Create a JSP incorrect login page.
  3. Create a JSP welcome page that says “Welcome [username].” Also have a link to logout, this should post to a logout servlet.
  4. Create a logout servlet that removes the username from the session if present, and sends the user back to the login page.

PROVE

Zip up your JSPs and servlets and submit them via I-Learn. You also need to submit the url to your website and a password and username combination that will allow the instructor to test your log in page. As a stretch challenge, you may complete the following steps for extra credit.

Stretch Challenge 1: Combine the incorrect login page with the sign in page. So that it is only one page, but the presence of extra data causes it to display an incorrect login message before the username and password boxes. This can be done by having the servlet store information on the request before transferring to the JSP.

Stretch Challenge 2: Deploy your app to one of your servers.

Stretch Challenge 3: Store a list of usernames and passwords in a file on the server and use that to validate in your login servlet.

If you complete any of the extra credit, please make sure to note it in your submission notes in I-Learn.
