TEAM READINESS ACTIVITY

PREPARE

Before doing this activity, each student should read the weekly preparation material to ensure that the team meeting is as efficient and effective as possible.

TEACH ONE ANOTHER

INSTRUCTIONS:

Step 1:

Create an HTML file that contains a form composed of the following components:

  • Name (text)
  • Email (text)
  • Major (radio button) with the following options:
    • Computer Science
    • Web Development and Design
    • Computer information Technology
    • Computer Engineering
  • Places you have visited (checkbox) with the following options:
    • North America
    • South America
    • Europe
    • Asia
    • Australia
    • Africa
    • Antartica
  • Comments (text area)

Step 2:

You do not need to style this HTML form, but each input should be labeled and separated from each other.

Step 3:

Create a PHP script to handle this form, so that when the form is submitted, it captures the form data and produces a web page that displays:

  • the user's name.
  • their email address, with a “mailto:” link for the email address.
  • their major.
  • a list of all the places they have visited.
  • their comments

Step 4:

Again, this data need not be styled, but it should be labeled and separated from each other.

Step 5:

Use the POST method for your form submission.

PROVE

Each student should submit a copy of the assignment to I-Learn. Please submit both the HTML form and the PHP script that handles this form. Note that these should be identical, but it is important to make sure that each student on the team has a copy of the work.
